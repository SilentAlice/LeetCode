# LeetCode Python Solutions

Python implementations of LeetCode problems.

## Structure

- `problems/` - Individual problem solutions
- `utils/` - Helper utilities and common data structures
- `tests/` - Unit tests for solutions

## Usage

Each problem file follows the naming convention: `lc{problem_number}.py`

Example:
```python
# lc1.py - Two Sum
def twoSum(nums, target):
    # Your solution here
    pass
```

## Running Tests

```bash
pytest tests/
```

## Requirements

See `requirements.txt` for dependencies.
